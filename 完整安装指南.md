# 完整安装和使用指南

## 第一步：安装Git

你的系统还没有安装Git，需要先安装：

### 安装步骤：

1. **下载Git**
   - 访问：https://git-scm.com/download/win
   - 点击 "64-bit Git for Windows Setup" 下载

2. **安装Git**
   - 运行下载的安装程序
   - 一路点击 "Next" 使用默认设置即可
   - 安装完成

3. **验证安装**
   - 关闭当前的PowerShell窗口
   - 重新打开PowerShell
   - 运行命令：`git --version`
   - 如果显示版本号，说明安装成功

## 第二步：配置Git（首次使用）

安装完Git后，需要配置你的用户信息：

```powershell
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"
```

## 第三步：创建GitHub仓库

1. 访问 https://github.com/new
2. 仓库名称输入：`douyin-sessionid-extractor`
3. 选择 Public
4. 不要勾选任何初始化选项（README、.gitignore等）
5. 点击 "Create repository"
6. 记下仓库地址，例如：`https://github.com/你的用户名/douyin-sessionid-extractor.git`

## 第四步：上传项目

### 方法A：使用自动脚本（推荐）

1. 双击运行 `upload_to_github.bat`
2. 脚本会检查Git是否安装
3. 如果未安装，会自动打开下载页面
4. 安装Git后重新运行脚本
5. 输入你的GitHub仓库地址
6. 等待上传完成

### 方法B：手动命令

打开PowerShell，运行：

```powershell
cd C:\Users\Administrator\.cursor\projects\empty-window

git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/douyin-sessionid-extractor.git
git branch -M main
git push -u origin main
```

## 第五步：等待自动构建

上传成功后：

1. 访问你的GitHub仓库页面
2. 点击顶部的 "Actions" 标签
3. 你会看到 "Build Android APK" 工作流正在运行
4. 等待15-30分钟构建完成

## 第六步：下载APK

构建完成后：

1. 在Actions页面，点击最新的构建记录
2. 向下滚动到 "Artifacts" 部分
3. 点击 "android-apk" 下载ZIP文件
4. 解压ZIP文件，里面就是APK

## 第七步：安装到手机

1. 将APK文件传输到Android设备
2. 在设备上：
   - 打开"设置" > "安全"
   - 启用"允许安装未知来源应用"
3. 点击APK文件安装
4. 打开应用
5. 点击"提取SessionID"
6. 授予ROOT权限
7. 查看提取结果
8. 点击"复制到剪贴板"

## 常见问题

### Q: Git安装后还是提示找不到？
A: 需要关闭并重新打开PowerShell窗口，让系统重新加载环境变量。

### Q: 推送时要求输入用户名密码？
A: GitHub已不支持密码登录，需要使用Personal Access Token：
   1. 访问 https://github.com/settings/tokens
   2. 点击 "Generate new token (classic)"
   3. 勾选 "repo" 权限
   4. 生成token并复制
   5. 推送时用token代替密码

### Q: 推送失败，提示权限问题？
A: 确保：
   - 仓库地址正确
   - 你有该仓库的写入权限
   - 网络连接正常

### Q: Actions构建失败？
A: 点击失败的构建记录查看日志，通常是依赖下载问题，重新运行即可。
